# 📚 BookSky – Book Management Web Application

## Overview

BookSky is a simple Book Management Web Application developed using HTML, CSS, and JavaScript. It allows users to add, display, and delete book records dynamically through an interactive user interface.

---

## Features

- Add new books with Title, Author, and Description
- Delete books dynamically
- Interactive popup form
- Responsive user interface
- Dynamic DOM manipulation using JavaScript

---

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Visual Studio Code
